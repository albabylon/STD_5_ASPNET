.Net Core:
1 Модульность - платформа состоит из модулей — отдельных компонентов, каждый из которых можно устанавливать и обновлять отдельно, 
используя менеджер Nuget.

2 Кроссплатформенность - возможность её работы под всеми основными операционными системами настольных ПК: Windows, MacOs и Linux

3 GUI - .NET Framework представлял возможность писать программы с интерфейсом для Windows, используя WinForms и WPF
Начиная с третьей версии, .NET Core поддерживает Winfowms и WPF, но только на Windows
.NET 6 будет добавлена технология .NET MAUI, которая будет поддерживать настольные и мобильные приложения Windows и MacOs.
Поддержкой настольных приложений Linux будет заниматься сообщество

Если вы хотите создать десктопное приложение с использованием .NET Core, 
для отображения пользовательского интерфейса отличным решением будет использование Electron JS.



ASP.NET Core — технология, работающая поверх кроссплатформенной среды .NET Core, и предназначенная для разработки именно веб-приложений.
Делать на ней можно как небольшие веб-сайты, так и крупные высоконагруженные порталы и сервисы, обрабатывающие огромное количество данных.

Для веб-приложений, которые раньше разворачивались в Windows-среде с помощью сервера IIS, 
теперь также доступна новая кроссплатформенная реализация сервера — Kestrel.


Что предлагает новый ASP.NET:
- Сам архитектурный паттерн MVC
- Технология создания веб-служб и сервисов Web API
+
- Была добавлена возможность более удобно интегрировать синтаксис HTML с кодом C# там, где нам это надо (с помощью так называемых тэг-хелперов)
- Значительно упрощена и сделана более удобной настройка конфигурации проекта
- Фреймворк теперь имеет свой встроенный контейнер для внедрения зависимостей, или IoC
